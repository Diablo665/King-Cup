# Сайт для создания кастомных карт
## Описание
Дайте волю фантазии — создайте свою версию «Кубка Короля»! Этот онлайн‑конструктор позволяет:

- придумывать необычные задания и правила для карт;
- комбинировать масти и значения по своему усмотрению;
- собирать уникальные колоды для вечеринок, корпоративов или семейных посиделок;
- использовать созданные карты для игры на сайте [King Cup](http://kingcup.ru/) 

## Технологии

- **React** — для создания пользовательского интерфейса
- **Redux** - для хранения данных в момент создания карт и управления ими
- **CSS Modules** — для стилизации компонентов
- **JavaScript/JSX** — для логики игры

## Как создать кастомные карты

1. Добавление новой карты:
- Масть карты: масть карты может быть любым символом, числом или слово.
- Номер карты: желательно указывать число от 1 до 13 (или J, Q, K для фигурных карт), но также можно указывать любой символ, число или слово.
- Текст карты: опишите действие или правило, связанное с картой.
- Есть правило: отметьте чекбокс, если карта вводит новое правило в игру (Если чекбокс отмечен, то при выпадании карты появится поле для ввода правила).
- Нажмите кнопку «Сохранить карту», чтобы добавить карту в коллекцию.

2. Просмотр и управление картами

В таблице отображается список всех добавленных карт с колонками:

После того как вы создадите нужное количество карт, можно будет скачать готовый файл, который потом использовать для игры. 

## Использование кастомных карт:

1. Зайдите на сайт.
2. Нажмите на кнопку "Загрузить свои карты".
3. Выберите файл с кастомными картами, который был создан ранее.
4. Начните игру.
